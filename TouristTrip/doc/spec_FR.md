Description globale : **Un voyage touristique. Un itinéraire créé pour visiter un ou plusieurs lieux d'intérêt (TouristAttraction/TouristDestination) souvent liés par un thème, une zone géographique ou un intérêt similaire à un type de touriste particulier. L'OMT définit le voyage touristique comme le voyage effectué par les visiteurs**.  
version : 0.2.0
